 #**Customer Retention & Churn Analysis Dashboard**

An interactive Power BI dashboard designed to analyze customer churn patterns and identify key factors affecting customer retention. This project helps businesses understand why customers leave and provides insights to improve retention strategies.

# Project Overview

Customer churn is a major issue for subscription-based businesses. This project focuses on analyzing customer data and visualizing patterns that lead to churn.
The dashboard provides a data-driven approach to:

- Identify at-risk customers
- Understand churn behavior
- Improve business decision-making
  
# Objectives
Identify customers likely to churn
Analyze key factors influencing churn
Provide actionable insights for retention
Support data-driven strategies

# Tools & Technologies
Power BI
Data Visualization
Business Intelligence
Data Analytics

# Key Performance Indicators (KPIs)
Total Customers
Churn Customers
Average Monthly Charges
These KPIs provide a quick snapshot of customer retention performance.

 # Dashboard Features & Analysis


 # Analysis:
Early-stage customers are most at risk
Retention strategies should focus on onboarding phase

# Key Insights
-  High churn is observed in new customers
-  Monthly contract users are more likely to leave
-  Customers with higher charges tend to churn more
-  Long-term customers show strong retention

# Conclusion

This dashboard successfully provides a clear and actionable view of customer churn behavior. It enables businesses to:
Predict customer behavior
Reduce churn rate
Improve customer satisfaction
